使用React学习开发一个PC端需要登录使用的平台。


使用到的技术栈：
react
react-router-dom
react-redux redux
redux-saga
axios
ant-design
bootstrap


实现部分：
使用ant-design搭建布局，使用react-router完成路由配置，并实现路由守卫。
使用redux完成，store配置，使用action， reducer，完成状态管理。
使用redux-saga完成异步的通过API获取数据
